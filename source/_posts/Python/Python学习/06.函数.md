## 认识函数
> 函数是**组织好的，可重复使用的**，用来实现特定功能的**代码段**

使用函数的好处：
* 函数可以重复使用，避免重复编写代码
* 提高代码复用性，提高开发效率

## 定义函数
```python
# 函数定义格式：
def 函数名(参数):
    函数体
    return 函数返回值
# def为定义函数的关键字，不可省略
# 定义函数
def add(x,y):
    result = x + y
    print(f"{x} + {y} 的结果是：{result}")
# 调用函数
add(1,2)
```
> 函数的参数和返回值不一定每个函数都需要，是可以选择的。但是函数必须先定义后使用。
* 函数定义中，提供的`x`和`y`，称之为：形式参数（形参），表示函数声明将要使用2个参数。
* * 参数之间使用逗号进行分隔
* 函数调用中，提供的`5`和`6`，称之为：实际参数（实参），表示函数执行时真正使用的参数值
* * 传入的时候，按照顺序传入数据，使用逗号分隔

注意：
* 函数的参数可以没有，也可以是任意个参数。

## 函数的返回值
> 函数的返回值，就是函数执行完毕之后，返回给调用者的结果。
> 调用有返回值的函数时，使用一个变量承接函数的返回值，随后就可以在函数外部使用这个变量来处理返回的数据。
```python
def add(x,y):
    result = x + y
    return result
# 使用一个变量承接函数的返回值
number=add(5,6)
print(number)
```
**注意**：
* 函数体在遇到`return`关键字时，就会结束函数的运行，并返回给调用者结果。所以写在`return`后面的代码不会被执行
* 函数如果没有返回值，则返回值默认为`None`,在某些特殊情况的处理下，也可以手动设置返回值为`None`
> 特殊的，`None`在布尔值判断时，会被认为是`False`
```python
def test1():
    return None

def test2():
    pass

print(test1())
print(test2())
```
## 函数的注释（说明文档）
> 函数的注释，就是为函数添加说明文档，便于其他人快速理解明晰函数的形参和返回值，函数的功能等。

函数的注释，一般使用`""" """`包裹，里面可以写多行注释，注释的内容会作为函数的说明文档。
```python
# 语法如下
def func(x):
    """
    函数整体说明
    :param x: 形参 x 的说明
    :return: 返回值的说明
    """
    函数体
    return 返回值
```
> **注意**注释内容位置应该在函数定义后，函数体之前。
