## 布尔类型和比较运算符
> 布尔类型`bool`，只有两个值：`True`和`False`。True本质上是`1`，False本质上是`0`。

定义变量存储布尔类型数据
`变量名称 = 布尔类型数据`

### 比较运算符
| 运算符        | 描述                   |示例              |
|:-----------|:---------------------|:------------------|
| ==         | 等于，满足为True，不满足为False | a==b              |
| !=         | 不等于，满足为True，不满足为False | a!=b              |
| >          | 大于，满足为True，不满足为False | a>b               |
| <          | 小于，满足为True，不满足为False | a<b               |
| >=         | 大于等于，满足为True，不满足为False | a>=b              |
| <=         | 小于等于，满足为True，不满足为False | a<=b              |
> **注意**：比较运算符的返回结果是布尔类型数据。

## 条件判断语句
> 条件判断语句，也叫分支语句，也叫选择语句。

条件判断语句一般分为三种：`if`、`elif`、`else`
* `if`：判断条件，满足条件执行if代码块，不满足条件执行else代码块。
* `elif`：判断条件，满足条件执行elif代码块，不满足条件执行else代码块。
* `else`：判断条件，满足条件执行else代码块，不满足条件执行else代码块。
* `if`、`elif`、`else`的判断条件都是布尔类型数据。
```python
age=input("请输入您的年龄：")
if age>=18:
    print('可以投票')
elif age>=16:
    print('可以参与选举')
else:
    print('不可以投票')
```
> 需要注意的是，判断语句是可以嵌套用来实现复杂逻辑下的条件判断功能的。
