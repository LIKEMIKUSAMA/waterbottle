本节主要介绍三种循环语句的用法和循环模式

<!-- more -->

## while循环
> 只要条件满足就会无限循环执行
> 循环条件中，True表示继续循环，False表示结束循环
```python
# 语法格式
while 条件:
    循环体
# 条件可以是变量也可以是函数，但是条件最终给出的结果一定要是布尔值，否则会报错
# 只要条件满足True就会一直循环输出 hello，直到条件为False结束
i = 0
while i<100:
    print("hello")
    i += 1
```

## for循环
> for循环是一种“轮询机制”，是对一批内容进行“逐个处理”的机制
> for循环一般用于遍历列表、元组、字典、字符串
```python
# 语法格式
 for 变量 in 被处理的数据:
     循环体

name = "for循环"
for i in name:
    print(i)
# 会把字符串的内容依次取出然后打印出来。
```
注意事项：
* 无法定义循环条件，只能被动取出数据进行处理
* 循环变量的取值范围是固定的，不能改变

## 循环中断
> 循环中断，就是跳出循环，结束循环，不再继续执行循环体

循环中断有两种方式
1、跳过当前循环，直接进入下一次循环，使用关键字 `continue`
2、退出循环体，直接中断循环。使用关键字 `break`
```python
for i in range(10):
    print("先执行")
    continue
    print("被跳过")

for i in range(10):
    print("先执行")
    break
    print("被中断")
```
循环中断的注意事项：
* 跳过当前循环，直接进入下一次循环，会继续执行下一次循环的循环体
* 退出循环体，直接中断循环，会结束循环体，不再执行循环体中的代码
* 不论是中断还是跳过循环，只会影响关键字**所属的内层循环**，对外层循环没有任何影响。
