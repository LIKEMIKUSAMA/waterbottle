## 字面量
> 代码中，被写在代码中的固定的值，称之为字面量

常见字面量
* 整数、浮点数、字符串均可成为字面量。

## 注释
> 在程序代码中对程序进行解释说明的文字

作用：注释不是程序，不能被执行，只是对程序代码进行解释说明，让别人可以看懂程序代码的作用，能够大大增强程序的可读性。
### 单行注释和多行注释
单行注释：
* 以`#`开头，`#`右边的所有文字当作说明，而不是真正要执行的程序，起辅助说明作用
* `#`号和注释内容一般以一个空格隔开。（官方建议格式）
* 一般用于对一行或一小部分代码进行解释。

多行注释：
* 以 一对三个双引号引起来(`"""注释内容"""`)来解释说明一段代码的作用使用方法
* 同时还支持换行。
* 一般用于对Python文件、类、方法进行解释，一般写在文件开头或方法开头的位置。

## 变量
> 程序运行时，用来记录数据或存储数据结果的抽象概念

变量的的定义格式：
`变量名称 = 变量的值`
* 每一个变量都有自己的名称，也称之为:变量名，也就是变量本身
* `=`号的作用时赋值，将右边的值赋给左边的变量
* 变量的值：代表变量自己存储的内容和数据。
* 变量的值是可变的

## 数据类型
> 三类主要的数据类型：字符串（`string`）、整型（`int`）、浮点型（`float`）。

验证数据类型可以使用Python自带的方法：`type()`
例如：`print(type(1551))`
使用type获取字面量数据类型，然后使用print输出。

### 数据类型转换
> 数据类型在特定场景下是可以相互转换的。如字符串转数字，数字转字符串等。

| 语句        | 说明          |
|:----------|:------------|
| int(x)    | 把x转为一个整数    |
| float(x)  | 把x转为一个浮点数   |
| str(x)    | 把x转为字符串     |

> 和`type()`一样,这三个语句都带有返回结果，可以使用变量存储然后`print`输出

**注意**：
* 任何内容都可转为字符串，但不是任何字符串都可以转为其他格式的内容。
* 浮点数转整数时，受限于精度，会导致小数部分丢失。

```python
num_str=str(11)
print(type(num_str),num_str)
```

## 标识符
> 编程时所以使用的一系列名字，用于标记文件、类、方法、变量等。

使用规则：（强制遵守）
1、命名中只允许使用英文、中文、数字、下划线（_）。其他任何内容都不允许
2、不推荐使用中文（注释除外）
3、任何标识符都不可以用数字开头
4、大小写敏感
5、不可使用Python自己保留的关键字。

变量的命名规范：（建议遵守）
1、见名知意，尽量做到看到名字就知道意义
2、多个单子组合作为变量，需要使用下划线作为分割。
3、变量中的英文字母全部小写。

## 字符串
> 字符串是Python中用于表示文本数据的一种数据类型。
### 定义方式
1. 使用单引号或双引号括起来，如：`'hello'`、`"hello"`
2. 三个单引号或三个双引号括起来，如：`'''hello'''`、`"""hello"""`
> 三引号定义法和多行注释写法一致，支持换行操作，使用变量接收就是多行字符串，不用变量接收就是多行注释

### 字符串的拼接
```python
# 单纯的字符串拼接
print('hello'+'world')
# 使用变量拼接
name='hello'
print(name+'world')
# 多变量拼接
tel='123456'
print(name+'world'+"手机号："+tel)
```
> 注意：非字符串格式是没有办法和字符串直接拼接的，需要先将非字符串格式转为字符串格式

### 字符串的格式化
#### 格式化方式一
```python
tel=123456
print('%s,%s'%('手机号是：',tel))
```
上述例子中的`%s`
* `%` 表示：这里有一个占位。
* `s` 表示：把变量变成字符串放入占位的位置。
> **注意**：多个变量占位，变量要用括号括起来，并按照占位的顺序填入。

| 格式符 | 转换                 |
|:----|:-------------------|
| %s  | 内容转为字符串放入指定位置      |
| %d  | 内容转为整数（10进制）放入指定位置 |
| %f  | 内容转为浮点型（小数）放入指定位置  |

#### 格式化方式二
```python
name='小明'
year=2019
high=168
print(f"姓名：{name}, 年龄：{year},身高：{high}")
```
使用`f-string`快速格式化字符串，使用`{}`括起来，里面可以放变量且不限制变量数据类型。

### 格式化的精度控制
使用辅助符号`m.n`来控制数据的宽度和精度
`m`，控制宽度，要求是数字(很少使用),设置的宽度小于数字自身时不生效
`.n`，控制小数点精度，要求是数字，会进行小数的四舍五入示例:
`%5d`:表示将整数的宽度控制在5位，如数字11，被设置为`5d`，就会变成:[空格][空格][空格]11，用三个空格补足宽度。
`%5.2f`:表示将宽度控制为5，将小数点精度设置为2小数点和小数部分也算入宽度计算。如，对11.345设置了%7.2f后，结果是:[空格][空格]111.35 。2个空格补足宽度，小数部分限制2位精度后，四舍五入为.35
