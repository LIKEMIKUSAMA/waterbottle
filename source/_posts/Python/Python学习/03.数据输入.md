本节主要介绍Python中数据输入的方式。

<!-- more -->
Pytnon中`Print`语句负责将内容输出到屏幕上。与之相反，`input()`语句负责从键盘输入数据。
使用也很简单：
* 使用`input()`可以从键盘获取输入
* 使用一个变量接收数据，获取到的数据默认是字符串格式，需要转换成对应的数据类型。
```python
name=input('请输入你的名字：')
print(name)
```
