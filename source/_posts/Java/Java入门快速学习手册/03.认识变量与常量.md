## Java常量与变量

在Java编程中，**常量（Constant）** 和 **变量（Variable）** 是存储数据的基本单元。它们的区别在于：

- **变量**：值在程序运行期间可以改变。
- **常量**：值在程序运行期间不可改变。

### 一、变量

#### 定义
变量是内存中的一个存储区域，其值可以在程序执行过程中发生变化。每个变量都必须声明其数据类型。
简单点理解：变量就是一个盒子，用来装程序运行中需要处理的数据。他的格式如下
```java
// 定义格式
// 数据类型 变量名称 = 数据;
int age = 20;
```
- 数据类型：用来限制盒子里存储的某种数据形式，你需要什么类型的数据就填什么。例如 `int`, `double`, `String` 等。
- 变量名称：变量的名称，用于标识变量。首字母小写，要有意义。
- 等于号：这里是赋值符号，用来将右边的值赋给左边的变量。
- 数据：变量存储的数据。

#### 特点
1. 变量在定义的时候可以不赋初始值，但是在**使用时变量里必须有值**。
2. 变量里的数据是可以被替换的。
3. 变量是什么类型，就必须赋值什么类型的数据。
4. 变量的作用域是定义变量的代码块。也就是在定义变量的开始，到他所属的这个代码块结束的范围内有效。如果不理解什么叫代码块没关系，我们后面会说。
5. 同一个作用域范围内，变量名称不能重复。
6. 变量的名称不能和Java的保留关键字重复。如果不清楚什么是关键字也没关系，我们会在后面说。

下面我们看一段代码示例
```java
/**
 * 变量的特点展示
 *
 * Version: 1.0
 * Author: 瓶子
 */
public class CH03VariablesAndConstants {
    public static void main(String[] args) {
        int a;          // 初始化但是不赋值
        String msg = "Hello World"; // 初始化并赋值
        System.out.println(msg); //  输出
        a = 5;  // 赋值
        System.out.println(a);  // 输出
        a= 10; // 改变赋值
        System.out.println(a);  //  输出

        //  赋值字符串，会报错
        //a = "123456"

        // int a;
        // 上面的int a; 是重复定义，会导致报错
    }
}
```

#### 命名规则
1. 变量名由字母、数字、下划线 `_` 或 `$` 组成。
2. 变量名不能以数字开头。
3. 变量名不能使用关键字（如 `int`, `class` 等）。
4. 变量名应具有可读性，通常使用小驼峰命名法（例如 `studentAge`）。

#### 示例代码
```java
/**
 * 变量的命名规则和使用
 *
 * Version: 1.0
 * Author: 瓶子
 */
public class CH03VariablesAndConstants {
    public static void main(String[] args) {
        int age = 20; // 声明并初始化一个整型变量
        double salary = 5000.50; // 声明并初始化一个双精度浮点型变量
        String name = "张三"; // 声明并初始化一个字符串变量

        System.out.println("姓名：" + name);
        System.out.println("年龄：" + age);
        System.out.println("工资：" + salary);

        age = 21; // 修改变量的值
        System.out.println("修改后的年龄：" + age);
    }
}
```


### 二、常量

#### 定义
常量是在程序运行过程中其值不会发生改变的量。在Java中，通过 `final` 关键字来定义常量。

#### 命名规则
1. 常量名通常使用大写字母。
2. 如果有多个单词组成，可以用下划线分隔（例如 `MAX_VALUE`）。

#### 示例代码
```java
/**
 * 常量的特点展示
 *
 * Version: 1.0
 * Author: 瓶子
 */
public class ConstantExample {
    public static void main(String[] args) {
        final double PI = 3.14159; // 声明一个常量PI
        int radius = 5;

        double area = PI * radius * radius; // 使用常量计算圆的面积

        System.out.println("圆的面积为：" + area);

        // PI = 3.14; // 编译错误，不能修改常量的值
    }
}
```


### 三、总结对比表

| 特征         | 变量                        | 常量                         |
|--------------|-----------------------------|------------------------------|
| 值是否可变   | 可以改变                    | 不可改变                     |
| 声明方式     | 直接声明数据类型和变量名    | 使用 `final` 关键字修饰      |
| 命名规范     | 小驼峰命名法（如 `age`）    | 全大写，多词用下划线分隔     |
| 使用场景     | 需要频繁更改的数据          | 固定不变的数据（如数学常数） |

### 四、注意事项
1. 在定义变量之前，必须先声明变量。
2. 常量通常定义在类或接口中，作为静态常量时使用 `static final`。
3. 常量有助于提高代码的可读性和维护性。

通过合理使用变量和常量，可以编写出结构清晰、易于维护的Java程序。
