## 一、流程控制基础
### 什么是流程控制？
>流程控制也可以称之为条件控制，是程序执行的核心机制。
> 完成条件控制的语句称为条件控制语句

条件语句主要包括三种基本结构：
* **顺序结构**：代码按从上到下的顺序依次执行。
* **选择结构**：根据条件决定执行路径（如 if-else、switch-case）。
* **循环结构**：重复执行某段代码（如 for、while、do-while）。

在现实生活中，我们经常会根据不同的情况做出不同的选择，比如天气下雨就带伞，天气晴朗就出门游玩。在 Java 程序里，条件语句也起着类似的作用，它能根据**条件判断的结果**，**决定执行**哪一段代码。

## 二、语句结构
条件语句有三种类型，每种类型都有对应的代码语句

### 顺序结构
> 代码按从上到下的顺序，依次执行，就是顺序结构。

例如
```java
/**
 * 顺序结构
 *
 * Version: 1.0
 * Author: 瓶子
 */
public class Main {
    public static void main( String[] args ) {
        System.out.println("1");    
        System.out.println("2");
        System.out.println("3");
    }
}
```
输出结果为
```java
1
2
3
```

### 选择结构
> 根据条件成立与否选择执行路径
> 大家经常听到的if-else语句就是选择结构中的一部分

#### if-else 语句
> 根据布尔条件执行不同代码块。

例如：
```java
/**
 * if-else伪代码
 *
 * Version: 1.0
 * Author: 瓶子
 */
public class Main {
    public static void main( String[] args ) {
        if (condition) {
            execute_this();
        }else {
            execute_that();
        }
    }
}

```
`if-else`语句在多个条件时，可以变化为`if-else if-else`结构。

#### switch-case 语句
> 本质上还是是`if-else if-else`语句，只是换了种表现形式，使用起来更加方便。
```java
public class Main {
    public static void main( String[] args ) {
        int type = (int)(Math.random() * 3);  //随机出0，1，2三个数字
        switch (type) {
            case 0:  //为 0 时执行
                captcha += (char)(Math.random() * 26 + 'a');  
                break;
            case 1:  //为 1 时执行
                captcha += (char)(Math.random() * 26 + 'A');
                break;
            case 2:  //为 2 时执行
                captcha += (int)(Math.random() * 10);
                break;
        }
    }
}
```
> **注意**：switch语句每一个分支选择结束后，需要添加一个`break`关键字作为分支选择结束的提示。不然语句会一直执行，直到遇到`break`后停下。

**实战样例**

使用if-else语句完成身体BMI指数计算
```java
import java.util.Scanner;
public class Main {
    public static void main( String[] args ) {
/**
 * 以千克为单位的体重除以以米为单位的身高的平方，得到BMI的值。针对20岁及以上的人群，BMI值如表所示
 * BMI             |      说明
 * BMI<18.5        |      偏瘦
 * 18.5<=BMI<25.0  |      正常
 * 25.0<=BMI<30.0  |      超重
 * 30.0<=BMI       |      过胖
 *
 * 编写程序，输入体重和升高，显示BMI值，并给出对应说明
 * Version: 1.0
 * Author: 瓶子
 */

        //获取体重和身高
        Scanner scanner = new Scanner(System.in);
        double weight, height, BMI;
        System.out.println("请输入体重，单位KG：");
        weight = scanner.nextDouble();
        System.out.println("请输入身高，单位米：");
        height = scanner.nextDouble();

        //计算BMI
        BMI = weight / (height * height);
        if (BMI < 18.5) {
            System.out.println("您的BMI值为" + BMI + "BMI值过低，体型偏瘦");
        } else if (BMI >= 18.5 && BMI < 25.0) {
            System.out.println("您的BMI值为" + BMI + "BMI值正常，请保持");
        } else if (BMI >= 25.0 && BMI < 30.0) {
            System.out.println("您的BMI值为" + BMI + "BMI值过高，体型超重，请控制体重");
        } else if (BMI >= 30.0) {
            System.out.println("您的BMI值为" + BMI + "BMI值极高，建议前往医院进一步检查。");
        }

    }
}
```

## 三、循环结构
> 循环用于重复执行代码，直到满足终止条件。所有的循环结构都可以使用其他循环语句来实现。
> 耳熟能详的for循环就是循环结构的一种。

### for 循环
> 适用于已知循环次数的场景。同时for循环中也可以再嵌套for循环。
语法结构如下
```java
for (初始化表达式; 条件表达式; 更新表达式) {
// 循环体代码
}
```
* 初始化表达式：用于初始化循环变量，只执行一次。
* 条件表达式：判断是否继续执行循环体，只要条件为true，就会执行循环体代码。
* 更新表达式：用于更新循环变量的值，每次循环结束后执行。

```java
/**
 * for循环样例
 *
 * Version: 1.0
 * Author: 瓶子
 */
// 基础for循环
public class Main {
    public static void main( String[] args ) {
        for (int i = 1; i <= 10; i++) {
            System.out.println(i);
        }
    }
}
//for循环嵌套计算 九九乘法表
public class Main {
    public static void main(String[] args) {
        for (int i = 1; i < 10; i++) {
            for (int j = 1; j <= i; j++) {
                System.out.print(j + "×" + i + "=" + (i * j) + "\t");
            }
            System.out.println();
        }
    }
}

```
在基础for循环这个例子中，`int i = 1`是初始化表达式，`i <= 10`是条件表达式，`i++`是更新表达式。

程序先初始化`i`为 `1`，然后判断`i <= 10`是否成立，若成立则执行循环体输出i的值，接着执行`i++`更新`i`的值，再进行下一次条件判断，直到`i <= 10`不成立时结束循环。

### while 循环
>while循环先判断条件，再执行循环体，只要条件为true，就会一直执行循环体代码。语法如下：

```java
while (条件表达式) {
    // 循环体代码
}
```
比如计算1到100的累加
```java
int sum = 0;
int i = 1;
while (i <= 100) {
    sum += i;
    i++;
}
System.out.println("1到100的累加和为：" + sum);
```
这里，先初始化`sum`为 `0`，`i`为 `1`，然后判断`i <= 100`是否成立。
若成立则执行循环体，将`i`累加到`sum`中并更新`i`的值，不断重复这个过程，直到`i <= 100`不成立。

### do-while 循环
>`do - while`循环与`while`循环类似，但它会先执行一次循环体，再判断条件。语法如下：
```java
do {
    // 循环体代码
} while (条件表达式);
```
同样以计算 1 到 100 的累加和为例：
```java
int sum = 0;
int i = 1;
do {
    sum += i;
    i++;
} while (i <= 100);
System.out.println("1到100的累加和为：" + sum);
```
> 在do - while循环中，无论条件表达式是否成立，循环体**至少会执行一次**。

## 四、循环控制语句
循环控制语句的作用就是用来控制循环的进度，决定循环是否继续。分为两种
* break：无论后面还有多少次循环，立即中止`break`所在的循环语句。
* countinue：跳过当前循环，无论后面还有多少代码没有执行。

### break语句
> 立即终止整个循环。

示例：当i=5时退出循环
```java
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // 退出循环
    }
}
```

### continue
> 跳过当前循环的剩余代码，直接进入下一次迭代。

示例：过滤特定条件的数据。只打印1-10中的偶数
```java
for (int i = 1; i <= 10; i++) {
        if (i % 2 != 0) {  // 可被2整除是偶数。
        continue;          // 跳过循环，不执行下面的输出语句。
        }
        System.out.println(i);
}
```
## 五、常见问题与注意事项
### 死循环
> 条件永远为真或未正确更新变量。

示例：while (true) 
```java
while (true) {
    System.out.println("一直输出内容");
}
```

## 六、应用场景示例
我们结合之前学习的方法和今天学习的条件控制语句，来完成一个实例。

大家可以先尝试不看代码自己写一下。功能的实现有很多种方式，代码仅供参考不是唯一解。
```java
/**
 * 循环与条件控制实例
 *
 * 1、随机生成1到100之间的整数1000个
 * 2、统计每个数出现的次数
 * Version: 1.0
 * Author: 瓶子
 */
import java.util.Arrays;
import java.util.Random;

public class MAIN {
    public static void main( String[] args ) {
        //获取生成好的1000个整数
        int[] randomnum;
        randomnum = randomNumbers();
        System.out.println("生成的随机数为：" + Arrays.toString(randomnum));

        //打印数字和出现次数
        numberCount(randomnum);
    }

    //1、生成1到100之间的整数1000个
    public static int[] randomNumbers() {
        //初始化random和numbers
        Random random = new Random();
        //加上[]代表初始化int类型的数组，不加代表初始化一个int类型的整数变量
        // int[1000]代表准备好1000个等待写入数字的行
        int[] numbers = new int[1000];
        //循环生成1000次数字
        for ( int i = 0; i < 1000; i++ ) {
            numbers[i] = random.nextInt(100) + 1;
        }
        return numbers;
    }

    //2、统计数字出现次数
    public static void numberCount( int[] array ) {
        //依次取出一个数，然后和1000个数字做对比，对比成功后计数器增加1，对比完成后输出结果
        //防止重复统计，增加一个控制开关，若当前数字在前面已经出现过，跳过不统计。
        for ( int i = 0; i < array.length; i++ ) {
            boolean isduplicate = false;
            int count = 0;
            int tmp = array[i];
            for ( int j = 0; j < array.length; j++ ) {
                //i>j，且两数相等，证明该数字已经再前面统计过一次，跳过该次统计。
                if ((tmp == array[j]) && (i > j)) {
                    isduplicate = true;
                    break;
                } else if ((tmp == array[j]) && (i <= j)) {
                    count++;
                }
            }

            if (isduplicate) {
                continue;
            }

            System.out.println("数字：" + tmp + " 出现的次数为：" + count);
        }
    }
}


```
