## Srting
### 什么是String
String类是Java中用于处理字符串的类。他的对象可以封装字符串数据，并提供了很多方法完成对字符串的处理。

### Srting的特点
* String是Java中一种特殊的字符串类。
* String类中的方法都是**线程安全的**。
* String类中的方法都是**不可变**的。
* String类中的方法都是**效率**比较高的。
* String类中的方法都是**final**的。
* String类中的方法都是**重写**的。

### String创建字符串对象的方式
1、直接使用双引号赋值对象封装
`String string = "hello";`
2、使用new关键字创建对象（使用构造器）
```java
String string1 = new String();  // 未赋值，默认值为空字符串
String string2 = new String("字符串");
```
3、使用String类中的静态方法创建对象
```java
char[] chars = {'h', 'e', 'l', 'l', 'o'};
String string3 = new String(chars);   // 使用字符数组创建字符串对象
System.out.println(string3);   // 输出 hello
```

### 创建对象的区别
* 以`"..."`创建出的字符串对象，会放在内存中的字符串常量池，相同内容的字符串只会保存一个。
* 以`new String()`创建出的字符串对象，会放在堆内存中，每次new都会新建一个对象，相同内容的字符串会保存多个。
```java
public class SrtingDemo {
    public static void main( String[] args ) {
        String s1 = "hello world";
        String s2 = "hello world";
        System.out.println( s1 == s2 );  // 输出：true
        
        String s3 = new String("hello world");
        String s4 = new String("hello world");
        System.out.println( s3 == s4 );  // 输出：false
    }
}
```

| toUpperCase()                                         | 把字符串转换成大写                                |

### String方法使用样例
```java
import java.util.Scanner;
/**
 * 字符串处理样例
 *
 * Version: 1.0
 * Author: 瓶子
 */
public class SrtingDemo {
    public static void main( String[] args ) {
        // 处理字符串数据
        String username = "admin";
        System.out.println("请输入用户名：");
        Scanner scanner = new Scanner(System.in);
        String inputUsername = scanner.next();

        if (inputUsername.equals(username)) {   // equals()只比较字符串内容，不比较地址
            System.out.println("true字符串一致");
        }
        if (inputUsername == username) {
            System.out.println("登录成功！");
        } else {
            System.out.println("false字符串不一致！");  // == 实际比较字符串地址，两次创建的字符串地址不同，所以输出false
        }

        // 修改字符串中间的内容
        String phonenumber = "12345678901";
        // 手机号隐藏中间4位数字
        //                           截取[0-3）位的数字    填充*号      截取7位之后的数字
        String newPhonenumber = phonenumber.substring(0, 3) + "****" + phonenumber.substring(7);  // 截取后拼接
        System.out.println(newPhonenumber);  //输出 123****901
    }    
}
```

## ArrayList
### 什么是ArrayList
> ArrayList是一种集合，用来装数据。ArrayList的底层实现是数组。
> ArrayList的底层实现是数组，所以ArrayList的查询性能比LinkedList高。

### ArrayList提供的常用方法
| 方法名                       | 功能                |
|:--------------------------|:------------------|
| add(Object obj)           | 向集合中添加元素          |
| add(int index,Object obj) | 向集合中添加元素，并指定添加的位置 |
| remove(int index)         | 删除指定位置的元素         |
| remove(Object obj)        | 删除指定元素，返回是否修改成功   |
| get(int index)            | 获取指定位置的元素         |
| set(int index,Object obj) | 修改指定位置的元素         |
| size()                    | 获取集合的长度           |
| isEmpty()                 | 判断集合是否为空          |
| contains(Object obj)      | 判断集合中是否包含某个元素     |
| clear()                   | 清空集合              |
| addAll(Collection c)      | 向集合中添加所有元素        |
| removeAll(Collection c)   | 从集合中移除所有元素        |
| retainAll(Collection c)   | 保留集合中指定的元素        |
| toArray()                 | 把集合转换成数组          |
| forEach(Consumer c)       | 遍历集合              |
| removeIf(Predicate p)     | 移除集合中的元素          |

### 创建ArrayList对象的方式
`ArrayList<String> list = new ArrayList<>();`
`<>`中的参数是泛型参数，带上泛型参数，可以保证集合中只能添加指定类型的元素，避免类型转换异常。
代码样例
```java
import java.util.ArrayList;

public class ArraryListDemo {
    public static void main( String[] args ) {
        ArrayList<String> list = new ArrayList<>();
        // 添加元素
        list.add("张三");
        list.add("李四");
        list.add("王五");
        list.add("赵六");
        // list.add(1);    // 生成实例时指定了泛型String，所以这里添加的元素非String会报错
        System.out.println("集合内容："+list);   // 输出 集合内容：[张三, 李四, 王五, 赵六]

        // 删除元素
        list.remove(0);
        System.out.println("删除元素后："+list); // 输出 删除元素后：[李四, 王五, 赵六]

        // 修改元素
        list.set(0,"张三");
        System.out.println("修改元素后："+list);  // 输出 修改元素后：[张三, 王五, 赵六]

        // 查询元素
        System.out.println(list.get(2));        // 输出 赵六
        // 遍历集合
        for ( int i = 0; i < list.size(); i++ ) {
            System.out.println(list.get(i));
        }
    }
}
```
