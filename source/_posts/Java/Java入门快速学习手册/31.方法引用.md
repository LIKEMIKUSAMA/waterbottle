## 一、静态方法引用
使用格式
`类名::静态方法`
使用场景
如果某个Lambda表达式里只是调用一个静态方法，并且`->`前后参数的形式一致，就可以使用静态方法引用。
```java
/**
 * 简化后Lambda表达式符合静态方法引用继续进行简化。
 *
 * Version: 1.2
 * Author: 瓶子
 */
// 父类
public class Test {
    public static void main( String[] args ) {
        Student[] students = new Student[3];
        students[0] = new Student("张三", 19, 170.0, '男');
        students[1] = new Student("李四", 21, 160.0, '女');
        students[2] = new Student("王五", 18, 180.0, '男');
        
        // 静态方法引用：           类名::方法名
        Arrays.sort( students, Student::compareByAge );
        
        // 输出排序后的数组
        for ( int i = 0; i < students.length; i++ ) {
            Student student = students[i];
            System.out.println(student);
        }
    }
}

class Student {
    // 姓名，年龄，身高，性别。
    private String name;
    private int age;
    private double height;
    private char sex;
    
    // 构造静态方法
    public static int compareByAge(Student o1, Student o2){
        return o1.getAge() - o2.getAge();
    }
    // 篇幅有限，省略构造器和getter和setter方法。
    // ...
}    
```

## 二、实例方法引用
使用格式
`对象名::静态方法`
使用场景
如果某个Lambda表达式里只是通过对象名称调用**一个实例方法**，并且`->`前后参数的形式一致，就可以使用实例方法引用。
```java
/**
 * 使用实例方法引用
 *
 * Version: 1.3
 * Author: 瓶子
 */
// 父类
public class Test {
    public static void main( String[] args ) {
        Student[] students = new Student[3];
        students[0] = new Student("张三", 19, 170.0, '男');
        students[1] = new Student("李四", 21, 160.0, '女');
        students[2] = new Student("王五", 18, 180.0, '男');

        // 实例方法引用对数组进行身高排序
        Student t = new Student();
        // 实例方法引用          实例名::实例方法
        Arrays.sort(students2, t::compareByHeight);
        
        // 输出排序后的数组
        for ( int i = 0; i < students.length; i++ ) {
            Student student = students[i];

        }
    }
}

class Student {
    // 姓名，年龄，身高，性别。
    private String name;
    private int age;
```

## 三、特定类型方法引用
使用格式
`特定类的名称::方法`
使用场景
如果某个Lambda表达式里只是调用一个特定类型的实例方法，并且前面参数列表中的第一个参数是作为方法的主调，后面的所有参数都是作为该实例方法的入参的，则此时就可以使用特定类型的方法引用。
```java
import java.util.Arrays;

public class Test2 {
    public static void main( String[] args ) {
        // 特定类型方法的引用
        // 需求：一个字符串数组，有一些人名，中文名称和英文都有，按照名字首字母升序排列
        String[] names = { "张三", "angolan", "张三", "Tom", "Jerry", "bobi", "曹操", "Mike", "Jack", "Rose","Anmi"};

        // 把这个数据进行排序  Arrays.sort(names,comparator)
        // Arrays.sort(names);   // 按照字符串的升序排序
        // 新要求：忽略首字母大小写进行排序（Java官方默认没有，需要我们自己指定比较规则）-> 匿名函数
//        Arrays.sort(names, new Comparator< String >() {
//                        @Override
//                        public int compare( String o1, String o2 ) {
//                            return o1.compareToIgnoreCase(o2);  //Java提供了字符串按首字母忽略大小写比较的方法。compareToIgnoreCase
//                        }
//                    }
//                   );
//        // 使用Lambda表达式，符合特定类型方法的引用
//        Arrays.sort(names,(  o1,  o2 ) -> o1.compareToIgnoreCase(o2) );
        // 特定类型方法引用 类型名称::方法名称
        Arrays.sort(names, String::compareToIgnoreCase );
        
        System.out.println(Arrays.toString(names));
    }
}
```

## 四、构造方法引用
使用格式
`类名::new`
使用场景
如果某个Lambda表达式里只是在创建对象，并且`->`前后参数的形式一致，就可以使用构造方法引用。
```java
import lombok.AllArgsConstructor;
import lombok.Data;
import lombok.NoArgsConstructor;

public class Test3 {
    public static void main( String[] args ) {
        // 理解构造器引用
        // 接口不能有对象，创建接口的匿名内部类对象。
//        TeacherInter t = new TeacherInter(){
//            @Override
//            public Teacher getTeacher( String name ) {
//                return new Teacher( name );
//            }
//        };

//        TeacherInter t =  name -> new Teacher( name );

        //构造器引用          类名::bew
        TeacherInter t = Teacher::new;

        Teacher teacher= t.getTeacher("张三");
        System.out.println(teacher);
    }
}
@FunctionalInterface
interface TeacherInter {
    Teacher getTeacher( String name );
}

@Data
@AllArgsConstructor
@NoArgsConstructor
class Teacher{
    private String name;
}
```
