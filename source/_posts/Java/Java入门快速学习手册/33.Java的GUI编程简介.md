## 什么是GUI编程
> GUI（Graphical User Interface）是图形用户界面，是一种用户界面，用户界面是计算机与用户进行交互的方式。
> GUI编程就是基于图形界面进行编程

### GUI编程的优缺点
* 优点
* * 界面友好，用户体验好
* * 与命令行界面相比，GUI界面更易操作，更易使用
* 缺点
* * Java制作GUI界面较繁琐
* * 目前企业界更多使用浏览器或其他语言编写的前端页面，Java主要用来编写后台程序。

### Java的GUI编程包
Java提供了两种GUI编程包
#### Swing（建议使用）
> 基于AWT衍生，提供了更加丰富的GUI组件，不依赖本地窗口系统，可跨平台使用。

常用组件

| 组件名称      | 功能                 |
|:-------------|:-------------------|
| JFrame       | 窗口                 |
| JPanel       | 面板(也叫容器，可用于组织其他组件) |
| JButton      | 按钮                 |
| JLabel       | 标签                 |
| JTable       | 表格                 |
| JTextField   | 文本框                |
| JTextArea    | 多行文本框              |
| JCheckBox    | 复选框                |
| JRadioButton | 单选框                |
| JComboBox    | 下拉框                |
| JList        | 列表框                |
| JScrollPane  | 滚动面板               |
| JOptionPane  | 对话框                |

#### AWT（已被抛弃）
> 提供原生的GUI组件，但是依赖于操作系统的本地窗口绘制，无法实现跨平台，跨系统使用。

### Swing编程样例
```java
import javax.swing.*;
/**
 * GUI编程样例
 *
 * Version: 1.0
 * Author: 瓶子
 */
public class GUIDemo {
    public static void main( String[] args ) {
        // 快速入门 GUI Swing的编程
        // 1、创建窗口，含用户名、密码输入框，登录按钮
        JFrame frame = new JFrame("登录界面");
        frame.setSize( 400, 300 );      // 窗口大小
        frame.setLocation( 200, 200 );      // 窗口位置
        frame.setDefaultCloseOperation( JFrame.EXIT_ON_CLOSE );     // 窗口关闭时退出程序
        frame.setVisible( true );   // 显示窗口

        JPanel panel = new JPanel();    // 创建面板（容器）
        frame.add( panel);  // 添加面板到窗口

        JButton button = new JButton( "登录");    // 创建按钮
        button.setBounds( 100, 100, 80, 20 );   // 按钮位置和宽高
        panel.add( button );    // 添加按钮
    }
}
```

### Swing还提供了几种布局管理器 
* FlowLayout 流式布局
> 它可以自动将组件放置在窗口上方横向排列，默认情况下，组件会自动换行。
* BorderLayout 边界布局
> 它可以将组件放置在窗口的北、南、东、西、中间位置，并且可以设置组件的填充方式。
* GridLayout 网格布局
> 它可以将组件放置在窗口的网格中，并设置网格的行数和列数。
* CardLayout 卡片布局
> 它可以将组件放置在窗口的卡片中，并设置当前显示的卡片。
