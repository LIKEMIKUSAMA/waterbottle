认识Java的输入输出与运算符。

<!-- more -->
在 Java 编程中，基本数据类型是构建程序的基石，而输入输出与运算符则是程序与外界交互、处理数据的关键工具。
掌握它们，能让你的程序 “听得懂” 外界指令，“算得清” 各类逻辑。本文将带你深入了解 Java 输入输出与运算符的用法、技巧与注意事项。

## 一、Java输入
Java 中常用的输入方式主要通过 `Scanner` 类和 `BufferedReader` 类实现，它们就像是程序的 “耳朵”，接收用户输入的数据。

### 使用 Scanner 类
> Scanner 类是最常用的输入方式，使用起来简单便捷，适用于大多数基础输入场景。 

#### 步骤详解：

1、使用 `Scanner` 类前，需要导入 `java.util.Scanner` 包。
使用 **`import java.util.Scanner;`** 来导入Scanner，使其在接下来的程序中可用。

2、然后创建一个 Scanner 对象：在主程序中创建一个 Scanner 实例，通常将标准输入流 System.in 作为参数传入，它代表从键盘获取输入。
`Scanner scanner = new Scanner(System.in);`

3、接着获取输入数据：根据需要获取的数据类型，调用 Scanner 类相应的方法。
详细代码如下
```java
/**
 * Scanner获取输入
 *
 * Version: 1.0
 * Author: 瓶子
 */
import java.util.Scanner;

public class CH04inputAndOutput {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        // 获取输入数据：根据需要获取的数据类型，调用 Scanner 类相应的方法。
        System.out.print("请输入一个整数：");
        int num = scanner.nextInt();  // 获取整数输入，输入非整数内容会报错

        System.out.print("请输入一个字符串：");
        String str = scanner.next();  // 获取字符串输入，遇到空格停止读取

        System.out.print("请输入一行文本：");
        String line = scanner.nextLine();  // 获取整行字符串，包括空格

        //关闭 Scanner：使用完毕后，应关闭 Scanner 对象以释放资源。
        scanner.close();
    }
}
```

### 使用 BufferedReader 类
> `BufferedReader` 类基于字符流进行输入，相比 `Scanner` 类，在处理大量文本输入时效率更高，且能更灵活地处理字符数据。

#### 步骤详解：
1、导入包：导入 `java.io.BufferedReader` 和 `java.io.InputStreamReader` 包。

2、创建 BufferedReader 对象：通过 InputStreamReader 将字节流转换为字符流，再创建 BufferedReader 实例。

3、获取输入数据：主要使用 readLine() 方法读取一行字符串，若要获取其他类型数据，需自行进行类型转换。

4、关闭 BufferedReader：使用完毕后，应关闭 BufferedReader 对象以释放资源。
> 使用 BufferedReader 类时，需要添加异常处理，防止程序因输入错误而崩溃。

以下是代码样例
```java
/**
* BufferedReader 获取输入
*
* Version: 1.0
* Author: 瓶子
*/
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;

public class CH04inputAndOutput {
   public static void main(String[] args) throws IOException, IOException {
      // 获取输入数据：使用 BufferedReader 类，使用try-catch块来捕获异常并抛出
      try{
         BufferedReader reader = new BufferedReader(new InputStreamReader(System.in));
         System.out.print("请输入任意内容：");
         String input = reader.readLine();   // 获取整行输入内容
         System.out.println("输入的内容是：" + input);
      } catch (IOException e) {
         System.err.println("读取输入时出错：" + e.getMessage());
      }

   }
}
```

## 二、Java 输出
> Java 的输出功能通过 System.out 实现，它就像是程序的 “嘴巴”，将处理结果展示给用户。

### print() 方法
> print() 方法用于将指定内容输出到控制台，输出后不会自动换行。
```java
System.out.print("Hello, ");
system.out.print("World!");
```

上述代码放到main中执行后，控制台输出 Hello, World! ，内容在同一行显示。

### println() 方法
> println() 方法同样用于输出内容到控制台，但输出后会自动换行。

```java
System.out.println("Hello");
System.out.println("Java");
```

执行结果为：
```
Hello
Java
```

### printf() 方法
> printf() 方法按照指定格式输出内容，类似于 C 语言中的 printf 函数，适用于格式化输出场景，如输出表格数据、控制小数位数等。
```java
double num = 3.1415926;
System.out.printf("圆周率保留两位小数为：%.2f", num);
```
输出结果为：
```
圆周率保留两位小数为：3.14
```

## 三、Java 运算符
> 运算符是 Java 进行数据运算和逻辑判断的关键工具，根据功能可分为算术运算符、赋值运算符、关系运算符、逻辑运算符、位运算符等。
### 算术运算符
> 算术运算符用于进行基本的数学运算，包括加（+）、减（-）、乘（*）、除（/）、取余（%）、自增（++）、自减（--）。
```java
/**
 * 运算符演示
 *
 * Version: 1.0
 * Author: 瓶子
 */
int a = 10;
int b = 3;
System.out.println("a + b = " + (a + b));  // 输出 13
System.out.println("a - b = " + (a - b));  // 输出 7
System.out.println("a * b = " + (a * b));  // 输出 30
System.out.println("a / b = " + (a / b));  // 输出 3，整数相除结果取整
System.out.println("a % b = " + (a % b));  // 输出 1，取余数
System.out.println("a++ = " + (a++));  // 先输出 a 的值 10，再将 a 自增为 11
System.out.println("++a = " + (++a));  // 先将 a 自增为 12，再输出 a 的值 12
```

### 赋值运算符
> 赋值运算符用于将右侧表达式的值赋给左侧变量，最基本的赋值运算符是 =，还有复合赋值运算符，如 +=、-=、*=、/=、%= 等。
```java 
int x = 5;
x += 3;  // 等价于 x = x + 3，x 的值变为 8
x *= 2;  // 等价于 x = x * 2，x 的值变为 16
```

### 关系运算符
> 关系运算符用于比较两个值的大小关系，结果为 boolean 类型（true 或 false），包括大于（>）、小于（<）、大于等于（>=）、小于等于（<=）、等于（==）、不等于（!=）。
```java
int m = 10;
int n = 20;
System.out.println("m > n = " + (m > n));  // 输出 false
System.out.println("m < n = " + (m < n));  // 输出 true
System.out.println("m == n = " + (m == n));  // 输出 false
System.out.println("m != n = " + (m != n));  // 输出 true
```

### 逻辑运算符
> 逻辑运算符用于对 boolean 类型的值进行逻辑运算，结果同样为 boolean 类型，包括逻辑与（&&）、逻辑或（||）、逻辑非（!）。
```java
boolean p = true;
boolean q = false;
System.out.println("p && q = " + (p && q));  // 逻辑与，全真才为真，输出 false
System.out.println("p || q = " + (p || q));  // 逻辑或，有真即为真，输出 true
System.out.println("!p = " + (!p));  // 逻辑非，取反，输出 false
```

### 位运算符
> 位运算符直接对二进制位进行操作，包括按位与（&）、按位或（|）、按位异或（^）、按位取反（~）、左移（<<）、右移（>>）、无符号右移（>>>）。
```java
int num1 = 6;  // 二进制为 0000 0110
int num2 = 3;  // 二进制为 0000 0011
System.out.println("num1 & num2 = " + (num1 & num2));  // 按位与，输出 2（二进制 0000 0010）
System.out.println("num1 | num2 = " + (num1 | num2));  // 按位或，输出 7（二进制 0000 0111）
System.out.println("num1 ^ num2 = " + (num1 ^ num2));  // 按位异或，输出 5（二进制 0000 0101）
System.out.println("~num1 = " + (~num1));  // 按位取反，输出 -7
System.out.println("num1 << 1 = " + (num1 << 1));  // 左移 1 位，输出 12（二进制 0000 1100）
System.out.println("num1 >> 1 = " + (num1 >> 1));  // 右移 1 位，输出 3（二进制 0000 0011）
```


## 四、运算符优先级与结合性
> **注意** 在一个表达式中存在多个运算符时，运算顺序由运算符的优先级和结合性决定。
> 优先级高的运算符先执行，优先级相同则根据结合性确定执行顺序。
> 例如，算术运算符优先级高于关系运算符，关系运算符优先级高于逻辑运算符。具体的优先级和结合性规则可参考官方文档。
> 但在实际编程中，为了避免歧义，建议使用括号明确运算顺序。
```java
int result = (2 + 3) * 4 / 2;  // 先算括号内，再依次进行乘除运算，结果为 10
```
