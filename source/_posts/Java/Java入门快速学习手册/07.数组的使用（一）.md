## 一、什么是数组
首先我们需要明确，**数组是一种用于存储固定数量、同类型元素的线性数据结构** 。
它通过数组名和一个整形下标可以访问整个数组中的每一个元素，例如`a[i]`,其中 `a` 是数组名，`i`是数组的整型下标。
Java是一种完全面向对象的编程语言，所以数组也是**对象**，这意味着数组的类型是引用类型，数组变量实际上是对数组的引用。而引用型变量在声明后**必须**通过实例化才能对变量进行访问。

我们总结出数组具有如下特点：
* **固定大小**：数组一旦创建，长度不可改变。
* **同类型存储**：所有元素必须是同一数据类型（如int[]只能存整数）。
* **连续内存**：元素在内存中连续存储，支持通过索引快速访问。
* **引用类型**：数组是对象，变量存储的是数组的引用（类似对象）

## 二、数组的声明与初始化
### 声明
数组的声明方式有两种，语法格式如下
```java
// 一维数组声明语法格式
{type} {arrayName}[];
// 或者
{type}[] {arrayName};

// 声明示例
int array[]; 
// 或
int[] array;
```
`type`为类型，可以是java中`任意数据类型`。包括基本类型和引用类型。
`arrayName`是数组的合法标识符，也就是数组的名字。
`[]`是数组的固定使用，用来区分声明的是普通变量还是数组。大家记得就好。
> 注意！数组声明后是不可以直接使用的，需要进行初始化后才可以使用，所以实际编程中，数组的声明，初始化，创建一般是同步进行。
 
### 初始化
> 初始化就是对已经创建（声明）好的数组进行赋值。
> 初始化又分为静态初始化与动态初始化。

#### 静态初始化
> 适合数组元素较少且初始元素可以枚举的情况。

如：
`int numbers[] = {1,2,3,4,5};`
`String[] days = {"Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"};`

#### 动态初始化
> 先指定数组的长度，然后再为数组元素赋值。语法格式为数据类型[] 数组名 = new 数据类型[长度]; 

例如，创建一个可以存储 5 个整数的数组：
```java
int[] numbers = new int[5]; // 创建一个长度为5，类型为int的数组。
// 为数组元素赋值
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;
```
> **注意！** 当使用动态初始化时，如果没有为数组元素赋值，数组元素会被赋予默认值：

| 元素类型    | 初始默认值        |
|------------|------------------|
| boolean    | False            |
| char       | `\u0000`（空字符）|
| 整数       | 0                |
| 浮点数     | 0.0              |
| 对象       | null             |

```java
// 不赋值，默认值为0
int[] array = new int[5]; // [0, 0, 0, 0, 0]
```

## 三、数组的常见操作
### 数组的访问与修改
> 数组中的每个元素都有一个**索引**（也叫下标），索引从`0`开始，通过索引可以访问和修改数组元素

例如前面的numbers数组，访问第一个元素（索引为0）的方式为 `numbers[0]`，修改第一个元素的方式为 `numbers[0]=60`
下面的代码展示了如何遍历数组并输出所有元素。
```java
int[] scores = {85, 90, 78, 92, 88}; // 静态初始化
for (int i = 0; i < scores.length; i++) {
    System.out.println("第 " + (i + 1) + " 个学生的成绩是：" + scores[i]);
}
//输出
第 1 个学生的成绩是：85
第 2 个学生的成绩是：90
第 3 个学生的成绩是：78
第 4 个学生的成绩是：92
第 5 个学生的成绩是：88
```
在这段代码中，`scores.length`用于获取数组的长度，通过`for`循环，利用索引`i`依次访问数组的每个元素并输出。

下面的代码展示了遍历数组并**修改对应的元素**，然后输出。
```java
int[] scores = {85, 90, 78, 92, 88}; // 静态初始化
for (int i = 0; i < scores.length; i++) {
    scores[i]+=10; // 修改数组元素
    System.out.println("第 " + (i + 1) + " 个学生的成绩是：" + scores[i]);
}
//输出
第 1 个学生的成绩是：95
第 2 个学生的成绩是：100
第 3 个学生的成绩是：88
第 4 个学生的成绩是：102
第 5 个学生的成绩是：98
```
在这段代码中，`scores[i]+=10`访问到数组中的元素，然后加10。

### 数组的排序
在实际应用中，经常需要对数组中的元素进行排序。Java 提供了`Arrays`类来方便地操作数组，其中`Arrays.sort()`方法可以对数组进行排序。
例如，我们对上面例子中的`scores`进行排序
```java
/**
 * 数组排序
 *
 * Version: 1.0
 * Author: 瓶子
 */
import java.util.Arrays;

public class ArraySortExample {
    public static void main(String[] args) {
        int[] scores = {85, 90, 78, 92, 88};
        Arrays.sort(scores);  // 排序
        for (int i = 0; i < scores.length; i++) {
            System.out.print(scores[i] + " ");
        }
    }
}
//输出
78 85 88 90 92
```
