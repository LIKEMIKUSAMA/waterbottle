## 一、内部类
> 一个类定义在另一个类的内部，那这个类就是内部类。

内部类共分4种：成员内部类、静态内部类、匿名内部类、局部内部类。
### 成员内部类
* 成员内部类不能有静态成员变量，不能有静态方法。
* 创建内部类的对象需要按特定格式：外部类名.内部类名 对象名 =  new 外部类名().new 内部类名()
* 创建完后就可以像正常类一样使用实例访问内部类的方法和成员变量。

构成格式如下
```java
public class Animals{
    public class Cat{  //Cat是Animals的成员内部类
        public void show(){
            System.out.println("成员内部类");
        }
    }
}
Animals.Cat cattest = new Animals().new Cat(); // 创建成员内部类的实例
cattest.show();
```

### 静态内部类
* 静态内部类有static修饰。属于**外部类本身拥有**
* 静态内部类和成员内部类一样，但是静态内部类不能访问外部类的非静态成员变量和方法。
* 创建静态内部类的对象格式：外部类名.静态内部类名 静态内部类对象名 = new 外部类名.静态内部类名();

```java
public class Animals{
    public static class Cat{  //Cat是Animals的静态内部类
        
    }
}
// 创建静态内部类的对象
Animals.Cat cattest = new Animals.Cat();
```

### 匿名内部类
> 匿名内部类是定义一个匿名类并创建对象，匿名内部类和成员内部类一样，但是匿名内部类不能有静态成员变量和方法。
> 匿名是指不需要主动为这个类声明名字，但是Java会默认给一个隐藏名。
> 匿名内部类格式：new 抽象类或接口名(){  }
> 本质是为了**更快的**创建一个子类对象。

#### 匿名内部类特点：
> 匿名内部类本质就是一个子类，并会立即创建出一个子类对象。

```java
// 匿名内部类结构
// new [需要继承的父类/接口名]() {
//     类体(一般就是方法重写)
// };

// 样例
// 父类
public abstract class Animal {
    public abstract void show();
}

// 子类继承重写父类
public class Cat extends Animal {
    @Override
    public void show() {
        System.out.println("正常子类重写父类抽象方法。");
    }
}

// 调用
public class Test {
    public static void main( String[] args ) {
        // 通常流程
        Animal a = new Cat();  // 生成子类实例
        a.show();  // 调用子类重写的方法。

        Animal b = new Animal() {  // 创建匿名内部类对象
            // 匿名内部类重写父类抽象方法。
            @Override
            public void show() {
                System.out.println("匿名内部类重写父类抽象方法。");
            }
        };
        b.show();  // 调用匿名内部类重写的方法。
    }
}
```
