文章目录规则

  • _posts 只放会进入文章流的技术文章
  • 一级目录固定为：linux、python、java、database、middleware、platform
  • 二级目录表示系列或细分专题，例如 quickstart-manual、centos、mysql

Front-matter 最小模板

资源规则

  • 截图、附件、脚本默认跟文章同级放在同名目录中
  • 不再新建散落的 source/file/... 或文章专用 source/img/...
  • 仅公共资源保留在 source/img/site/ 和 source/downloads/shared/

组织原则

  • 目录解决“物理放哪儿”
  • categories 解决“站内怎么聚合”
  • tags 只做补充检索，不再承担主分类职责
